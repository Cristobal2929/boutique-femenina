Este repositorio contiene la página web de una boutique femenina de ropa para eventos en La Línea de la Concepción.  
La web es una única página HTML autocontenida, sin dependencias externas más allá de Google Fonts.  
Todas las imágenes se encuentran en la carpeta `static/img/` y se usan según el orden solicitado.  
Para publicar, basta con servir los archivos estáticos en cualquier servidor web.
